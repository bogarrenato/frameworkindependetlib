# Owned Brand Pack

This registry item copies a starter CSS override for the `owned-brand` brand key.

Suggested import order:

1. `@fuggetlenfe/tokens/contract.css`
2. `@fuggetlenfe/tokens/figma-preset.css`
3. `./owned-brand.css`

After the files are copied into the consuming app they are fully consumer-owned.
Edit colors, radius, typography, and spacing there without changing the upstream component package.
