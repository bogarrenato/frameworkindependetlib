# React Custom App

Ez a React app ugyanazt a `@fuggetlenfe/react-wrapper` libraryt fogyasztja, mint a Brand 2 demo, de a design mar egy masik CSS librarybol jon:

```tsx
import { FfButton } from '@fuggetlenfe/react-wrapper'
import '@fuggetlenfe/tokens/contract.css'
import '@fuggetlenfe/brand-styles/custom-brand.css'
```

Inditas:

```bash
pnpm dev:react:custom
```
